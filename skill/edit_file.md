适用系统：macOS/Linux  
触发关键词：修改文件、编辑文件、替换文本、插入内容、删除行、批量替换  
必须流程：  
1) 先读取本技能文件后再生成命令  
2) 修改前必须先读取目标文件内容并确认修改范围  
3) 路径和文件名都要加双引号，避免空格导致失败  
4) 默认先备份原文件，再执行修改  
5) 优先使用可重复执行、可审计的命令（如 sed、perl、awk），避免交互式编辑  
6) 修改完成后必须再次读取关键片段，确认结果并返回变更说明  

命令模板：  
- 备份文件：cp "文件路径" "文件路径.bak"  
- 替换整文件中的文本（macOS sed）：sed -i '' 's/旧文本/新文本/g' "文件路径"  
- 按行号查看区间：sed -n "起始行,结束行p" "文件路径"  
- 删除指定行（macOS sed）：sed -i '' 'N d' "文件路径"  
- 在指定行后插入（macOS sed）：sed -i '' 'Na\
插入内容
' "文件路径"  
- 批量替换（目录内指定后缀）：find "目录路径" -type f -name "*.后缀" -exec sed -i '' 's/旧文本/新文本/g' {} \;  

失败兜底：  
- 若目标文件不存在，返回失败并说明路径不存在  
- 若权限不足，返回失败并提示权限问题  
- 若替换后无变化，明确提示“未匹配到目标文本”  
- 若语法导致修改失败，回滚备份文件并返回错误信息  

约定修改位置：默认在 edit 目录下执行；修改后备份文件保存在同目录下，命名为 "文件名.bak"。
