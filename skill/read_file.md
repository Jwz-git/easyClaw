适用系统：macOS/Linux  
触发关键词：读文件、查看文件内容、cat、read file、预览文件  
必须流程：  
1) 先读取本技能文件后再生成命令  
2) 路径和文件名都要加双引号，避免空格导致失败  
3) 默认优先安全预览（前 200 行），避免一次性输出超长内容  
4) 用户指定行号范围时，按范围读取；未指定时按分段读取  
5) 默认不修改任何文件，只执行只读查询命令  

命令模板：  
- 查看文件前 200 行：sed -n "1,200p" "文件路径"  
- 查看指定行范围：sed -n "起始行,结束行p" "文件路径"  
- 查看全文（小文件）：cat "文件路径"  
- 查看尾部 200 行：tail -n 200 "文件路径"  
- 获取总行数：wc -l "文件路径"  
- 判断是否为文本文件：file "文件路径"  

失败兜底：  
- 若文件不存在，返回失败并说明路径不存在  
- 若是二进制文件，提示为非文本内容，不直接输出乱码  
- 若权限不足，返回失败并提示权限问题  
- 若文件过大，按 200 行分块返回并提示继续读取下一段  

约定读取位置：默认当前目录相对路径；用户给绝对路径时按绝对路径执行。
